#Завдання:

Створити консольну програму мовою Python і написати клас Interview який міститиме:
1. Поля імя інтервювера, тривалість інтервю, результат інтервю
2. Три додаткових поля, які найкраще описують даний клас	
3. Конструктор (із вказаними дефолтними значеннями)
4. Деструктор
5. Метод, що повертає стрічкове представлення класу (__str__())
6. Статичне поле
7. Статичний метод, що повертає значення статичного поля
8. В main() методі визначіть 3 об’єкти типу із завдання (з-за допомогою передачі різної кількості параметрів) та виведіть інформацію про них з-за допомогою методу з пункту 4 в консоль
9. Розроблений код має бути залито в гіт репозиторій як pull request! В репозиторії мать бути README і .gitignore файл. 

#Інструкція із запуску:
1. Відкрити термінал у папці, куди ви хочете завантажити файли завдання
2. Скопіювати репозиторій за допомогою команди git clone https://github.com/max-dmytryshyn/Python-Labs.git
3. Перейти у папку з файлами за допомогою команди cd Python-Labs
4. Перейти на гілку Lab_1 за допомогою команди git checkout Lab_1
5. Запустити програму за допомогою команди python3 main.py
